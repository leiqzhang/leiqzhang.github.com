wget下载整个网站的镜像

$ wget —mirror -p —convert-links -P ./LOCAL-DIR WEBSITE-URL

  • –mirror : 下载镜像
  • -p : 下载gif等显示html必须的资源
  • –convert-links : 下载后，将文档中所有的链接进行转换，以支持本地查看
  • -P ./LOCAL-DIR : 需要保存下载文档的目录
